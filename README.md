# Atualização do Sistema — Visualização Individual de Equipamentos

Esta atualização implementa a possibilidade de visualizar, de forma detalhada, um equipamento específico a partir do seu ID.

## O que foi adicionado
- Nova funcionalidade para consulta individual de equipamentos.
- Expansão do modelo com os seguintes atributos:
  - **Modelo**
  - **Código**
  - **Número de Patrimônio**

## Benefícios
- Acesso rápido e organizado a informações específicas.
- Documentação mais completa dos equipamentos.
- Melhoria no fluxo de inspeção e manutenção.
